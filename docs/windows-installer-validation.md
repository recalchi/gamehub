# GameHub Windows Installer Validation

Date: 2026-05-30  
Repository: `https://github.com/recalchi/gamehub`  
Branch: `main`

## Stack and installer decision

- App stack: Electron + React + TypeScript (`electron-vite`).
- Installer choice: `electron-builder` with `nsis` target (Windows x64).
- Update model: `electron-updater` consuming GitHub Releases artifacts.

## Current installer defaults (v1)

- Product name: `GameHub`
- Version source: `package.json` (`0.1.0`)
- Architecture: x64
- Scope: per-user (`%LocalAppData%\Programs\GameHub`)
- NSIS mode: assisted (`oneClick=false`)
- Data policy: preserve user data on uninstall (`deleteAppDataOnUninstall=false`)
- Installer languages: `pt_BR`, `en_US`

## Build and release commands

```powershell
npm run dist:win
npm run dist:win:portable
npm run dist:win:full
npm run publish:win
```

## Expected artifacts

- `release/GameHub-Setup-x64-<version>.exe`
- `release/GameHub-Setup-x64-<version>.exe.blockmap`
- `release/GameHub-portable-x64-<version>.zip`
- `release/latest.yml`
- `release/releases.json`
- `release/GameHub-Setup-x64-<version>.exe.sha256`
- `release/GameHub-portable-x64-<version>.zip.sha256`

## Silent operations

```powershell
.\GameHub-Setup-x64-<version>.exe /S
.\GameHub-Setup-x64-<version>.exe /S /D=%LocalAppData%\Programs\GameHub
"%LocalAppData%\Programs\GameHub\Uninstall GameHub.exe" /S
```

## Security checks

- Updater does not use `git pull`.
- Updater runs only in packaged builds.
- Updater rejects downgrades (`allowDowngrade=false`).
- Update hosts are allowlisted in `src/main/core/updater.ts`.
- Manifest + SHA256 are generated by `scripts/generate-update-manifest.mjs`.

## Signature readiness

Build does not require certificates. Production signing can be added with:

```powershell
signtool sign /fd SHA256 /tr http://timestamp.digicert.com /td SHA256 /a release\GameHub-Setup-x64-<version>.exe
signtool verify /pa /v release\GameHub-Setup-x64-<version>.exe
```
